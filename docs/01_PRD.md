# LITAN Platform PRD

LITAN 是中文同人商品預購與現貨販售平台。

## 核心需求
- 中文介面
- 粉紫漸層風格
- 可愛狸貓元素
- 首頁分成預購專區與現貨專區
- 預購與現貨完全分流
- 老師管理 Teacher Master
- Teacher ID 使用 4~6 碼短 UUID
- 一位老師底下管理多個商品
- 商品使用 Tags，不使用商品分類
- 商品類型：預購 / 現貨
- 預購保留匯款與補款
- 現貨使用賣貨便付款，不需匯款
- 預購無庫存
- 現貨有庫存
- 已到貨預購商品才可合併出貨
- 現貨可合併現貨，但不可與預購合併
- 訂單編號格式 LT000001
- A4 四分之一出貨單
- Excel 匯入匯出
- 投票系統
- 老師排序
